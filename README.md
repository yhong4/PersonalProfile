# PersonalWebsite

## Technology stack
CSS3, ReactJs, ReactRouter, JSX, Awesome icon

## Browser
Chrome

## Responsive
No

## Project URL
[Link](https://personal-profile-hy.herokuapp.com/home)

## install in localhost
1. Install npm
```
brew install node
```

2. Install all dependencies
```
npm install 
```

3. Run project and enjoy
```
npm run start
```

4. The web will run at 
```
http://localhost:3000
```
